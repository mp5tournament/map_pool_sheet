# 图池表生成器

图池表生成器，使用本项目的[发布版](https://mp5tournament.github.io/map_pool_sheet/)，可以根据填入的图池bid列表生成相应的图池表。图池表格式可以自行定义，内置提供了MP5的选图表和主表格格式。
生成后的内容可直接复制进入Excel、QQ表格等当中。
